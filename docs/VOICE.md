# Product voice (Courier UI)

**Language:** English (native)  
**Tone:** Cold War / 1950s civic propaganda — warm, plain, a little wry  
**Audience:** General public — **no crypto jargon**

## Preferred lexicon

| Instead of (tech) | Use (plain / era) |
|-------------------|-------------------|
| Identity / account | **Papers** |
| Recovery phrase | **Passport** (word list) |
| One-shot invite / link | **Safe-conduct** |
| Conversations / chats | **Letters** |
| Fingerprint / public key | **Seal** / **mark** |
| Display name | **Cover name** |
| Ephemeral / disappearing | **Burn after reading** |
| Dead man’s switch | **Dead man’s switch** (keep — plain enough) |
| Decoy vault | **Cover story** |
| E2E encrypted | **Sealed** (“only the two of you can read this”) |
| Relays / Tor / P2P / queues | *(don’t say in UI)* |
| Directory / search users | **No phone book** |
| Wipe / purge | **Burn** |
| Public Square | **Public Square** |
| Video area | **Kinotek** |

## Rules

1. If a grandmother wouldn’t know the word, rewrite it.
2. Flavor words sparingly — don’t turn every label into a novel.
3. Humor OK; sneer at the user never.
4. Technical truth lives in docs — not in button labels.
