## Licence (à trancher)

Suggestion pour un repo public “outils de résistance” :

| Licence | Quand la choisir |
|---------|------------------|
| **GPL-3.0** | Tu veux que les forks restent open source |
| **AGPL-3.0** | Idem + usage réseau (relais / services) aussi contaminé |
| **Apache-2.0 / MIT** | Adoption max, y compris forks propriétaires |

Tant que ce n’est pas décidé, le README indique `License: TBD`.
